# Week 7: JavaScript Best Practices To-Do App

## 📌 Project Overview

A refactored To-Do List application built using JavaScript best practices.
The app demonstrates clean code structure, state management, and data persistence using localStorage.

---

## 🚀 Features

* Add, delete, and toggle tasks
* Filter tasks (All, Active, Completed)
* Clear completed tasks
* Persistent data using localStorage
* Clean and maintainable code structure

---

## 🧠 Concepts Covered

* DOM Manipulation
* Event Delegation
* State Management
* localStorage Persistence
* Code Refactoring
* JavaScript Best Practices

---

## 📂 Project Structure

* index.html → Structure
* style.css → Styling
* main.js → Logic

---

## 🛠️ How to Run

1. Clone the repository
2. Open index.html in your browser
